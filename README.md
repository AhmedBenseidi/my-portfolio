# 💻 Professional Portfolio Website (Laravel & Filament)

This is the source code for my professional portfolio. A high-performance, SEO-optimized web application built to showcase projects, skills, and professional experience.

---

## 🌟 Key Technical Features

- **Multi-Language Core:** Dynamic switching between **Arabic, English, and French**.
- **SEO Optimized:** Full integration with `artesaos/seotools` for dynamic Meta tags and OpenGraph support.
- **Admin Dashboard:** Powered by **FilamentPHP v3** for easy management of projects and skills.
- **Modern UI:** Responsive design built with **Tailwind CSS** and **Blade**.
- **Cloud Ready:** Optimized for deployment on **Railway** or any Linux-based server.

---

## 🛠️ Technology Stack

| Component          | Technology                                |
| :----------------- | :---------------------------------------- |
| **Framework**      | [Laravel 12](https://laravel.com)         |
| **Admin Panel**    | [FilamentPHP v3](https://filamentphp.com) |
| **Frontend**       | Tailwind CSS / Alpine.js / Livewire       |
| **Database**       | MySQL                                     |
| **Deployment**     | Railway.com                               |
| **OS Environment** | Xubuntu Linux                             |

## 🚀 Getting Started (Local Development)

To run this project locally on your machine, follow these steps:

### Prerequisites

- PHP 8.2 or higher
- Composer
- Node.js & NPM
- MySQL

### Installation

1. **Clone the repository**
